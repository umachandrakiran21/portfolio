# Portfolio Website

A personal portfolio website built using HTML, CSS, and JavaScript to showcase skills, projects, education, and contact information.

## Features

* Responsive portfolio layout
* Education section
* Skills section
* Projects section
* Contact section
* Interactive buttons using JavaScript
* Hover effects and modern styling
* CSS Flexbox and Grid layouts

## Technologies Used

* HTML5
* CSS3
* JavaScript
